通讯录管理系统

项目简介
基于 C++ 实现的控制台通讯录管理系统，支持联系人的增删改查与数据管理。

技术栈
- C++
- STL（字符串）

核心功能
- 添加联系人（姓名/性别/年龄/电话/地址）
- 显示所有联系人
- 按姓名查找联系人
- 按姓名删除联系人
- 按姓名修改联系人信息
- 一键清空通讯录

数据结构
结构人 {
 字符串名称;// 姓名
 智力性;// 性别：1-男，2-女
 国际年龄;// 年龄
 国际电话;// 电话
 字符串附加器;// 地址
};

struct地址簿{
 人称 arr[1000];// 最多存储1000人
 智力大小;// 当前人数
};

运行方式
使用 Visual Studio 打开 '.cpp' 文件，编译运行即可。

代码量
200+ 行，模块化设计，无野指针、内存越界问题。
